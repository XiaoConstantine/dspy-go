try - Test an Optimizer#

Run an optimizer with a built-in dataset.

Usage:

Arguments:

ArgumentDescriptionRequired
<optimizer>Optimizer name (bootstrap, gepa, mipro, simba, copro)Yes

Flags:

FlagShortDescriptionDefault
--dataset-dDataset to use (gsm8k, hotpotqa)gsm8k
--max-examples-nNumber of examples to use10
--verbose-vVerbose outputfalse
--timeout-tTimeout in seconds300
--output-oOutput file for resultsstdout
--jsonOutput as JSONfalse

ArgumentDescriptionValues
<profile>Use case profilespeed, balanced, quality, custom

Flags (for custom profile):

FlagDescriptionRange
--time-budgetTime budget in seconds1-3600
--quality-targetTarget quality score0.0-1.0
--complexityTask complexitylow, medium, high
--data-sizeDataset sizesmall, medium, large
